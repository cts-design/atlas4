###Getting Started

All from root directory

`git clone https://github.com/JosephShering/Atlas4.git`

`cd Atlas4`

`npm install`

`bower install`

`grunt server`

If you want to activate live reload for the project instead, run

`grunt server watch`


Go here:
`https://localhost:8000/#/user/list`
